# Data files

This project used the supplementary tables from Kang and Kim 2024.

Expected raw data files:

```text
data/raw/Table S2.xlsx
data/raw/Table S3.xlsx
```

Table S2 contains TF PWM/PSSM motif data.  
Table S3 contains enhancer variant sequences and MPRA expression values.

These files were used to build the Kang 2024 XML input from the `kim2013.xml` template.
